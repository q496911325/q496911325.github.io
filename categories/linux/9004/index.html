本系列首页路径：linux就该这么学

一. Vim文本编辑器

  1. 三种模式
    命令模式：控制光标移动，可对文本进行复制、粘贴、删除和查找等工作
    输入模式：正常的文本录入
    末行模式：保存或退出文档，以及设置编辑环境

  2. 命令模式中常用命令

    命令作用
    dd将标准输出重定向到一个文件中(清空原有文件的数据)
    5dd将错误输出重定向到一个文件中(清空原有文件的数据)
    yy将标准输出重定向到一个文件中(追加到原有内容的后面)
    5yy>将错误输出重定向到一个文件中(追加到原有内容的后面)
    n将标准输出与错误输出共同写入到一个文件中(清空原有文件的数据)
    N将标准输出与错误输出共同写入到一个文件中(追加到原有内容的后面)
    u将标准输出与错误输出共同写入到一个文件中(追加到原有内容的后面)
    p将标准输出与错误输出共同写入到一个文件中(追加到原有内容的后面)

  3. 末行模式中可用的命令

    命令作用
    :w保存
    :q退出
    :q!强制退出(放弃对文档的修改内容)
    :wq!强制保存退出
    set nu显示行号
    set nonu不显示行号
    :命令执行该命令
    :整数跳转到该行
    :s/one/two将当前光标所在行的第一个one替换成two
    :s/one/two/g将当前光标所在行的所有one替换成two
    :%s/one/two/g将全文中的所有one替换成two
    ?字符串在文本中从下至上搜索该字符串
    /字符串在文本中从上至下搜索该字符串

五. 编写SHELL脚本

  1. 接收用户的参数

    命令作用
    $0当前shell脚本程序的名称
    $#总共有几个参数
    $*所有位置的参数值
    $?显示上一次命令的执行返回值 0表示没有错误 其他任何值表明有错误
    $nn=1...9
    $$这个脚本程序的PID
    $@跟$*类似，但是可以当作数组用

  2. 文件测试语句

    命令作用
    -d测试文件是否为目录类型
    -e测试文件是否存在
    -f判断是否为一般文件
    -r测试当前用户是否有权限读取
    -w测试当前用户是否有权限写入
    -x测试当前用户是否有权限执行

  4. 整数值比较语句

    运算符作用
    -eq是否等于
    -ne是否不等于
    -gt是否大于
    -lt是否小于
    -le是否等于或小于
    -ge是否大于或小于

  5. 字符串比较语句

    运算符作用
    =比较字符串内容是否相同
    !=比较字符串内容是否不同
    -z判断字符串是否为空
