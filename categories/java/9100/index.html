JavaSE轻松入门基础视频教程

一. Java语言介绍

1.Java的特性
面向对象、可移植性(JVM)、健壮性、分布式、多线程

2.Java的工作方式
java源文件(.java格式结尾)—经过java编译器的编译—字节码文件(.class结尾)—>被类装载器装载到java虚拟机(JVM)—被JVM解释给操作系统—操作系统来执行

3.Java开发系统的构成
Java虚拟机(JVM)：指令集、寄存器、堆栈、垃圾处理器、方法区域组成
类库：标准类库(由官方提供，可以直接使用类库中的类)、开发者自己的类
包：本质上是文件夹的形式，用于组织项目文件。
JRE：Java运行时环境(Java Runtime Environment)
JDK：Java开发套件(Java Development)

4.Java技术的应用
JavaME：目前市场份额很小，IOS/Android现在是主流
JavaSE：标准版，用于桌面软件的编程
JavaEE：为企业级开发提供一整套的解决方案
