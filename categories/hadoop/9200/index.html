大数据开发hadoop精讲

一. hadoop模块介绍

  1. hadoop模块
    common：工具类 rpc通信框架
    hdfs：hadoop分布式文件存储系统 主从架构 主进程：namenode(1个) 从进程：datanode(多个) 助理进程(secodarynamenode) 分担namenode压力
    yarn：集群资源调度框架 主从架构 主进程：resourcemanager(负责统筹资源) 从进程：nodemanager
    mapreduce：分布式计算框架

二. hadoop的搭建

搭建准备：
1）IP配置
vim /etc/udev/rules.d/70-persistent-net.rules
vim /etc/sysconfig/network-scripts/ifcfg-eth0
2）主机名
hostname 主机名
vim /etc/sysconfig/network
3）主机映射
vim /etc/hosts
4）关闭防火墙和selinux
service iptables stop
chkconfig iptables off
vim /etc/selinux/config(改为disabled )
5）将系统的启动级别改为3
vim /etc/inittab
6）创建普通用户，并为普通用户添加sudolers权限
useradd
passwd
vim /etc/sudoers 用户名 ALL=(ALL) NOPASSWD: ALL
7）配置免密登录
1）生成秘钥
ssh-keygen
2）发送秘钥
ssh-copy-id
8）安装jdk
vim /etc/profile

export JAVA_HOME=路径名
export PATH=$JAVA_HOME/bin:$PATH

source /etc/prifile

9）时间同步
1）不能联网 搭建时间服务器
2）能联网 找一个公网中的共用的时间服务器
ntpdate 公网时间服务器地址
10）安装版本选择 2.7.6
